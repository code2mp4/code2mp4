Open Source Project
Code2MP4
Agent-native video production pipeline
Why it exists
AI agents can write code. Now they can write videos.
Black-box text-to-video gives you an opaque MP4. Traditional editors require a human. Code2MP4 gives coding agents a structured, editable, deterministic video pipeline.
How it works
The workflow layer for agent-generated video
1
Storyboard
Structured JSON — scenes, goals, visuals, motion specs
2
Motion Source
Editable HTML/CSS/GSAP — the source code of your video
3
Quality Check
7-dimension review — AI-phrase detection, platform fit
Get started
Three commands to start
git clone https://github.com/code2mp4/code2mp4
pnpm install && pnpm dev

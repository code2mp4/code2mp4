# Product Launch — Script System
> Category: Marketing
> Duration: 15-45s
> Scenes: 4-5

## Narrative Arc
Hero → Feature 1 → Feature 2 → Specs → CTA

## Information Density
- ONE feature per scene
- Product name visible in hero AND CTA scenes
- Feature names: 6-12 characters max
- Feature descriptions: 15-25 characters max
- Specs: number + label format

## Pacing Map
| Section | Time | Duration | Feel |
|---------|------|----------|------|
| Hero | 0-8s | 8s | Cinematic reveal — product first, text delayed 0.5s |
| Feature 1 | 8-16s | 8s | Confident — product shot + one-line benefit |
| Feature 2 | 16-24s | 8s | Contrast — different angle or use case |
| Specs | 24-32s | 8s | Data — 3 specs in grid, staggered 0.15s |
| CTA | 32-40s | 8s | Direct — one command + URL |

## Voice & Tone
- Confident, aspirational, warm
- Product is the hero — minimize brand copy
- Features describe WHAT, not HOW
- Emotional benefit > technical spec
- "Available now" not "Coming soon"

## Hook Patterns
1. **Product Reveal** — Product fades/zooms in from darkness, no text for 0.5s
2. **Close-up Detail** — Tight shot of a detail (texture, edge, logo)
3. **Context Shot** — Product in use, then title overlay
4. **Lighting Drama** — Warm key light sweep across product surface

## Scene Templates
### 1. Hero (0-8s)
- Product visual centered, isolated
- Brand name top or logo lockup
- Headline: product name + one-line descriptor
- Subhead: tagline or category
- Lighting: dramatic, single source, warm

### 2-3. Features (8-24s)
- Each: feature name (accent color) + benefit (body color)
- Visual: product detail shot, lifestyle context, or UI screenshot
- Layout: visual left + text right, or text bottom third
- Stagger: feature 2 enters 0.3s after feature 1 exits

### 4. Specs (24-32s)
- 3 key specs in a horizontal grid
- Format: number (large, monospace) + label (small, muted)
- Optional: comparison with previous model
- Cards enter staggered with back.out(1.4)

### 5. CTA (32-40s)
- Full product shot OR logo lockup
- ONE command statement
- URL or price visible
- Subtle scale pulse on CTA text (0.3s, ease: elastic)

## Anti-Slop Checklist
- [ ] Product name visible in hero AND CTA
- [ ] No fake specs or invented features
- [ ] Features describe benefits, not implementation
- [ ] Lighting is intentional (not default flat)
- [ ] CTA is a single, clear action
- [ ] No more than 1 feature per scene
